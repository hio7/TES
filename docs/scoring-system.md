# TES Scoring System

## Symbols and Meaning

| Symbol | Meaning |
|--------|---------|
| 10     | Red ● clean hit |
| 01     | White ◆ clean hit |
| X      | Double hit |
| T      | Time-up |
| P.A    | Intentional after-blow (-1 Point, -1 Clean Hit)

- Points include all hits.
- Clean Hits are single, uncontested strikes.
- Penalties do not affect match result but impact award scoring.
